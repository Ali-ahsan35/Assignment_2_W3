# OwnerDirect

OwnerDirect is a responsive front-end website project built using pure HTML, CSS, and JavaScript.  
The project focuses on a clean hero section design with a mobile swipe slider and a dedicated desktop hero image layout.

This project follows a **mobile-first approach** and does not use any external libraries or frameworks.
